# webpack3-ejs-project

> 一个用webpack3构建，并且使用ejs模板开发的项目

## 构建步骤

### 全局安装webpack
``` bash
(sudo) npm install -g webpack@3
```

### 全局安装webpack-dev-server
``` bash
(sudo) npm install -g webpack-dev-server@2
```

### 安装项目依赖
``` bash
npm install
```

### 开发环境
``` bash
npm run dev
```

### 生产环境
``` bash
npm run build
```
